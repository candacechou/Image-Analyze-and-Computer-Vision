# Image Analysis and Computer Vision
This is the lab exercises on DD2423 Image Analyze and Computer Vision on KTH Royal Institute of Technology in Stockholm.

There are 3 Labs in this course:

1. Filtering operations
    - Continuous and discrete Fourier Transform
    - Multiplication, Scaling, Rotation, Information in Fourier phase and magnitude.
    - Gaussian Filtering
    - Smoothing, subsampling

2. Edge detection & Hough transform
    - Differential Geometry based Edge Detection
    - Extraction of Edge Segment
    - Hough Transform
    
3. Image segmentation
    - K-means
    - Normalize-cut
    - mean shift
    - graph cut
